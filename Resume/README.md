## Projects

### 1. [Centralized Report Server](./01-centralized-report-server.md)
**Impact**: 80% reduction in report retrieval time | 90% reduction in manual effort

A unified reporting platform built with Python and Flask that consolidated fragmented reporting systems across teams. Implemented automated onboarding through CI templates, enabling real-time updates and significantly improving cross-team visibility.

**Key Metrics:**
- 80% faster report retrieval
- 90% less manual onboarding effort
- Real-time reporting enabled
- Multi-page responsive web application

**Skills**: Full-stack development, System design, CI/CD automation, Performance optimization

---

### 2. [AI Models Evaluation Framework](./02-ai-models-evaluation.md)
**Impact**: Identified 10+ OWASP security vulnerabilities | Production-ready LLM evaluation

Developed a comprehensive LLM evaluation framework using DeepEval to assess security vulnerabilities specific to banking use cases. Created custom modular classes and high-level design documentation enabling continued framework development.

**Key Metrics:**
- 10+ OWASP vulnerabilities mapped
- Custom modular evaluation framework
- Business requirements successfully integrated
- Comprehensive design documentation delivered

**Skills**: AI/ML security, Threat modeling, Framework design, Security assessment, Python development

---

### 3. [Finacle API Automation Framework](./03-finacle-api-automation.md)
**Impact**: 90% reduction in testing time | 124 bugs identified and 80% resolved

Engineered a Java-based automation framework for Finacle banking APIs with intelligent test data generation. Framework reduced manual testing overhead and improved system reliability through comprehensive test coverage.

**Key Metrics:**
- 90% faster testing execution
- 70% less time on data preparation
- 124 bugs identified
- ~80% bug resolution rate

**Skills**: API automation, Java development, Test data management, Quality assurance, Banking domain expertise
