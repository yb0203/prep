## What Each Article Contains

Each article in the `articles/` directory includes:

```
[article-name]/
├── README.md
│   ├── Source Information (URL, title, topic)
│   ├── Article Overview (what it covers)
│   ├── Reading Session Structure (how we'll learn)
│   ├── Topics to Cover (checklist)
│   └── Notes & Key Learnings (filled during session)
│
└── session_transcript.md
    ├── Reading Session Details
    ├── Session Content (filled during reading)
    ├── Key Takeaways (filled as we progress)
    ├── Questions Asked (recorded during session)
    └── Next Steps
```
