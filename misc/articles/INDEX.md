## How to Use the /read Command

### Starting a Reading Session
```
/read system-design-core-concepts
```

### During a Reading Session
- **Ask Questions:** "What does this mean?" or "Can you explain this further?"
- **Continue:** "Continue reading"
- **Pause:** "Pause here, let me think about this"
- **Review:** "Summarize what we've learned so far"
- **Test:** "Test my understanding"

### After Completion
- Check `session_transcript.md` for full conversation record
- Notes and key learnings are stored in the article directory
- You can start a new session anytime or continue from where you left off

## Adding New Articles

To add a new article for reading sessions:

1. Create new directory: `articles/[article-name]/`
2. Create `README.md` with article overview
3. Create `session_transcript.md` for conversation tracking
4. Add entry to this INDEX.md file
5. Use `/read [article-name]` to start
